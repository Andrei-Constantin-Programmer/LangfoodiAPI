What is C4?

The C4 model is a technique for creating the diagrams that represent a system. There are four of them, all starting with the letter 'c' (hence the name):

  • Context diagram - shows the system and its relationships with users and other systems
  • Container diagram - decomposes the system into interconnected containers (applications or storage systems)
  • Component diagram - decomposes a container into interconnected components (modules/layers)
  • Code diagram - class diagram
